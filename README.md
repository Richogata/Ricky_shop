# Ricky_shop - Shopify Theme

Theme Shopify (Online Store 2.0) prêt à importer.

## Import rapide
1. Compresse le dossier en `.zip`.
2. Shopify Admin > Boutique en ligne > Thèmes.
3. `Ajouter un thème` > `Téléverser un fichier zip`.
4. Publie le thème et personnalise les sections depuis l'éditeur.

## Structure incluse
- Home avec Hero, collection mise en avant, produit vedette, texte.
- Pages produit, collection, panier, page standard.
- Header + footer configurables.

Ensuite, envoie-moi la liste de tes produits et je les prépare pour intégration.
